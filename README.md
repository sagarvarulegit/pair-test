# pair-test
## <u>How to run the code </u>
### 1. Clone the repository
### 2. Run the code
`mvn exec:java -Dexec.mainClass="com.example.MajorityElement"`
### 3. Run the tests
`mvn test`
### Tests are located at src/test/java/com/example/MajorityElementTest.java
